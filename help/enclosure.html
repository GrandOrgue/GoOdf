Enclosures

Enclosures change the volume of ranks mounted on a windchestgroup which references that enclosure.

The main parameter associated with an enclosure is Amplitude Minimum Level (integer 0-100, required) volume, if the enclosure is closed. The units are linear amplitude , not dB.

It's also possible to set the MIDIInputNumber so that the enclosure can be automatically connected to what an user has set as initial MIDI configuration in GrandOrgue.
